What is AI Chat Widget?

AI Chat Widget is a universal, open-source chatbot solution that can be integrated into any website. It works with multiple AI providers including OpenAI, Claude, GigaChat, YandexGPT, Ollama, and any OpenAI-compatible API.

üîå Universal AI Provider

üíæ Flexible Storage

üìö Knowledge Base

Simply drop .md or .txt files into a folder. No database required!

üåê Page Context Aware

Bot understands where it's working (URL, title, content, selected text).

üì± Telegram Alerts

üöÄ Easy Integration

Quick Start

Add the widget to your website with just one line of code:

<script src="https://your-server.com/widget/widget.js"
        data-api="https://your-server.com"
        data-theme="light"
        data-position="bottom-right"
        data-title="AI Assistant">
</script>
<link rel="stylesheet" href="https://your-server.com/widget/widget.css">
